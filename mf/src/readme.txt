This project uses the following compiler verions:
- Haskell version: 7.10.2-a
- uuagc-0.9.52.1
- uhc-light
To build the package it is necessary to follow the following instructions:
- Go to the src directory in a bash console; In this directory there is file called "Build.sh"
- Type in the console "./Build.sh". This script compiles all necesarry code.


Commands:


function
To execute an analyze about an example, you need to enter the following command:
run functionName "Examplename". 
For example you could write: 
run cp "slv/example1"
